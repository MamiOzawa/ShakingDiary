# ShakingDiary
**授業で作った日記アプリ**

- 100文字の日記機能
- スマホをふる強さでその日の嬉しさ/悲しさ/怒りの感情の強さを表現して記録する機能
- グラフ化して月毎に振り返る機能
- ツイート機能（ツイッターAPIキーは各自で取得し，teitter4j.propartyに記述してください）
